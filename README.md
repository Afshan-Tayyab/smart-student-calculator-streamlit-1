# 🧮 Smart Student Calculator

A simple web-based calculator built using Python and Streamlit.  
This project allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division through an interactive web interface.

## 🚀 Live Application
https://smart-student-calculator-app-1-j4kmyljvuc8exqxmuqlsea.streamlit.app/

## 📌 Features
- Enter two numbers
- Choose an operation (+, -, *, /)
- Get the calculated result instantly
- Handles division by zero errors

## 🛠 Technologies Used
- Python
- Streamlit

## ▶️ How to Run the Project

1. Clone the repository  
git clone https://github.com/Afshan-Tayyab/smart-student-calculator-streamlit-1.git

2. Go to the project folder  
cd smart-student-calculator-streamlit-1

3. Install dependencies  
pip install -r requirements.txt

4. Run the Streamlit app  
streamlit run app.py
